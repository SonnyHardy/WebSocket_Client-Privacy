Privacy Policy

Last updated: June 2025

STOMP WebSocket Client is a browser extension designed to allow users to connect to WebSocket servers using the STOMP protocol. We take your privacy seriously and this page outlines how we handle your data.

1. Data Collection

The extension does not collect, track, or store any personal information. All WebSocket configurations, messages, and tokens are stored locally in your browser.

2. Authentication Data

If you use JWT or OAuth2 tokens, they are stored only in your browser’s localStorage for the purpose of maintaining sessions. These tokens are never transmitted to any third-party server or collected by the extension author.

3. Message History

Message history (sent/received) is optionally stored in your browser's localStorage. You have full control to clear or export this data.

4. Internet Access

The extension connects to WebSocket servers that you specify. It does not send or receive any data on its own unless you initiate a connection.

5. Data Sharing

No data is ever shared with any third party.

6. Future Updates

If future versions of the extension include features such as user accounts or payment integration, this privacy policy will be updated accordingly. You will be informed via the Chrome Web Store page.

7. Contact

If you have any questions or concerns, please contact the developer at: sonnydev265@gmail.com

Thank you for using STOMP WebSocket Client.
